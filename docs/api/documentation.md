# Documentation Endpoint

## Overview

The API service can host the static MkDocs site under `/documentation` when the generated `site/` directory is present.

## Endpoint

- **URL**: `/documentation`
- **Method**: `GET`
- **Authentication**: Not required

## Request

No parameters or request body.

## Response

Returns the static HTML documentation site generated by MkDocs.

## HTTP Status Codes

| Code | Description |
| :--- | :--- |
| 200 | Documentation site is available and served. |
| 404 | Documentation site is not mounted (for example, `site/` is missing). |

## Example Request

```bash
curl -I "https://api-livekit-vyom.indusnettechnologies.com/documentation"
```

## Notes

- This route is mounted only if the local `site/` directory exists at service startup.
- Build docs with MkDocs before expecting this endpoint in fresh environments.
